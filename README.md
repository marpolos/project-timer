# Criando um timer multifuncional
![Alt text](https://media4.giphy.com/media/eQJj6BJIW13vS5cBXj/giphy.gif?cid=ecf05e479x6hclj0bcvsi1xgv0h2fucre0l4gan52xjwtg2k&rid=giphy.gif)

Esse projeto visa a construção de um contador de tempo simples para ser usado nos momentos de intervalo entre uma atividade e outra ou para descanso.
Quando fui personalizar encontrei a imagem do Lula molusco e decidi usar a paleta de cores do gif dele. Usei a extensão colorZila do Chrome para pegar as cores.

## Sugestões
- Estou aberta a sugestão na implementação do timer, o principal desafio é decidir se devo tocar uma música tema durante a passagem do tempo ou não.
- Foi bem desafiador fazer o contador de acordo com a hora, inicialmente não usei o tempo local para contar e criei 3 campos para hora, o que não deu certo. Descobri um hook personalizado o useHour e o utilizei, porém pretendo implementar uma lógica para essa contagem.

## Em desenvolvimento:
- Aba para hora local;
- Aba para contagem regressiva;
- Routes: tela principal, contagem, hora local e contagem regressiva.
